See No Evil

Link

Role: Solo Dev
Tools: Unreal Engine 5

See No Evil was made for my 3D Game Design class in Unreal Engine 5, using Blueprints. It's a first-person, survival horror game with invisible monsters.

While the monsters themselves are not directly visible, the ways they affect the world are. For example, any lights near a monster will begin to flicker on and off. The player can use this to determine their rough location.

I had a lot of fun working in Unreal and fleshing out my idea for the game into a vertical slice. At some point, I would like to come back and redo all the code in C# instead of Blueprints.

Make Me a Sandwich!

Link

Role: Programmer
Tools: Godot

Make Me a Sandwich! was made ~48 hours for my school's annual Game Jam. My two teammates did all of the art for the game, while I did all the programming in GDScript.

I also did the level design and mechanic design.

I think my best code was for the spiderweb bounce pads in the cabinet section. I used enums and exported variables to streamline the level design process by making variables like launch speed and launch angle directly editable from the viewport.

I'm extremely happy with the final product considering it was my first game jam. Although, I think I could've been better at implementing all the art my teammates made, as a few assets didn't make it into the final submission.

WASTED

Link

Role: Solo Dev
Tools: Godot

Wasted was made for my 2D Game Design class using Godot, and primarily coded in GDScript (I used GLSL for the shaders).

The idea was to create the modern equivalent of Drunken Master (1978) starring Jackie Chan, if you've ever seen that.

I had a lot of fun making the drunk effect shader using GLSL. It scales with the amount the player drinks, which caused sudden jumps in the effect after every drink. I fixed it by interpolating between the old value and the new value whenever the drink count is updated.

This was my first experience with a modern game engine, so most of time with this project was spent translating what I knew from my Computer Science classes in High School into Godot.
